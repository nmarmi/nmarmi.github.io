Some of my favorite projects

01 • GitHub

Quantum ML

Problem: quantum state tomography is expensive — full reconstruction requires exponentially many measurements as system size grows.

Stack: Python, Neural Networks

Outcome: predicted unknown reduced density matrices from partial measurements, cutting tomography cost and accelerating quantum simulations.

02 • DraftIQ • DataAPI

DraftIQ + Player Data API

Problem: fantasy baseball drafts lack real-time player analytics and a reliable data API that teams can build on.

Stack: JavaScript, Node.js, Next.js, MongoDB Atlas, Git

Outcome: full-stack app and licensed API handling 1000+ concurrent requests with average response time under 100ms, built by a team of 4 using Agile.

03 • GitHub

Playlister

Problem: build a full-stack playlist manager with a redesigned backend, improved database abstraction, and reliable unit test coverage.

Stack: JavaScript, Node.js, React, MongoDB

Outcome: clean client-server architecture with modular database layer and unit tests validating core functionality end-to-end.

04 • GitHub

Clip Extractor

Problem: manually cutting video sub-clips by face is slow, error-prone, and doesn't scale to large video libraries.

Stack: Python, face-recognition, OpenCV, MoviePy, Click, Git

Outcome: automated face-based extraction with batch processing reduced manual editing to nearly 0%, with a lightweight CLI for reproducibility.

05

Algorithmic Trading Program

Problem: translating trading strategies from research into a system that can be backtested, extended, and validated reliably.

Stack: Python, Polars, TA-Lib, Git

Outcome: modular backtesting framework validated through paper trading, achieving consistent positive returns with only 20% losing trades.

Listening Engine • Capco
